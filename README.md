# ELDENRING-BOSSTRACKER

## Description
This university project was developed as part of the module **Interactive Systems**. It is a companion application for Elden Ring, a Souls-like action RPG, designed to help players track defeated major enemies (bosses) and monitor their overall progression. The application also functions as a focused wiki, providing structured information about these bosses. 

Due to time constraints and predefined project goals, some features are only partially implemented or simulated to demonstrate the intended final functionality. In the current version, some example wiki pages are filled with content, bosses can be marked as defeated and will then appear grayed out in the interface and there are various filter options that help users quickly find specific bosses, although the search bar is not yet functional. 

The boss dataset is based on the open-source [ELDEN RING API](https://eldenring.fanapis.com/) project. The data was cleaned, corrected, and extended, and a dedicated server was implemented to avoid repeated external API calls.
